Install Network Tools

This will install essential network analysis tools on all Ubuntu hosts including:

  • Network discovery: nmap, traceroute, mtr, arp-scan, fping
  • Traffic analysis: tcpdump, wireshark, tshark
  • Monitoring: iftop, nethogs, vnstat
  • Performance: iperf3, netperf, speedtest-cli

This operation requires sudo access on target hosts
